Examples

There are a few examples included in the sdk. On these pages, they are described step by step. For each plugin, there's a Delphi and a C++ page.

  • FruityGain is an effect plugin.
  • Osc3 is an generator plugin. It's a full generator, which means that it implements Gen_Render. If it had been a hybrid generator, it would have implemented Voice_Render instead.
  • Internal Controller is also an effect plugin. It doesn't process any sound, but instead it allows you to control other controls (in FL Studio).
  • Sine is a simple plugin that plays a sine wave. It demonstrates how to implement per-voice level and pan and slides.

The FruityGain pages contain the most information. The Osc3 and Sine pages assume you've read the FruityGain pages and build on from that knowledge, only explaining the things specific to a generator plugin.

There's no written documentation for the Internal Controller example yet (except for the comments in the sourcecode).

Note that these examples are quite old and don't show recent features.
